What is it?

KunLauncher was created to be an easy way to share applications in development with testers. Instead of zipping files and sending them over Google Drive, Discord, or MediaFire, instead they could just be downloaded from the Kun Launcher with a single click. KunLauncher has the ability to automatically update both itself and the applications it hosts, giving it the ability to carry out its purpose with ease.

Why is the download process retarded?

You might have noticed that this website is hosted with 000webhost. 000webhost is a great hosting service for people who are learning or can't afford web hosting, both categories we fall into. Technically, there is no "release" file of the KunLauncher provided. Instead, the download page simply supplies a link to the file the Launcher itself downloads in order to perform updates. This file is named `latest.noexe` because 000webhost does not allow executable files to be hosted on their platform. This is likely to keep security tight, and we understand the reasoning. So we rename the file to latest.noexe and then just rename it when updating. Unfortunately, if you have to do this process yourself to download the launcher for the first time, it looks quite shady. Compare that with the fact that the executable is not signed (and Windows likes to complain about this), as well as a closed source, and the whole program looks quite shady. This is a necessary compromise, unfortunately, and right now there's nothing we can really do about it.

Why another launcher? Why not use Steam?

because I'm poor and Steam is expensive. I needed a quick way to get my applications to people without using Google Drive or MediaFire. if I updated a piece of software and had to send it over to someone, chances are they wouldn't replace the old one with it. this lead to me looking at my friend's desktops over Skype screenshare and seeing at least ten different bobwars folders, which made me very sad. with kunlauncher, i just have to say hit the update button and run the test again, and it's good. if i could put my shit on steam, trust me, I would. unfortunately, it's $100 per piece of software put on steam. maybe one day i could do something like Facepunch Prototypes, but that's still $100. for now, enjoy the fact that KunLauncher does not require an account or typical installation. just run the file and delete it all when you're done.
